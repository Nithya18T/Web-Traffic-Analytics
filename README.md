# Web Traffic Analytics

## Objective

Analyze website traffic data to understand user behavior and engagement patterns.

## Features

* Page view analysis
* Session duration analysis
* Bounce rate analysis
* Traffic source evaluation
* Data visualization
* User engagement insights

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Structure

```bash id="ab4r7m"
Web-Traffic-Analytics/
│
├── data/
│   └── website_traffic.csv
│
├── outputs/
│
├── src/
│   └── analysis.py
│
├── requirements.txt
├── README.md

```

## How to Run

```bash id="o0vk7u"
pip install -r requirements.txt
```

```bash id="3t5yff"
python src/analysis.py
```

## Output

The project generates:

* Traffic source charts
* Session duration graphs
* Bounce rate visualization
* Exit page analysis
* Device type analysis

All graphs are automatically saved in the `outputs` folder.
